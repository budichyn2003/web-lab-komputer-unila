# Changelog

All notable changes to the LabKom Unila website will be documented in this file.

## [1.0.0] - 2024-01-01

### Initial Release

#### Features
- ✨ Modern dark-themed responsive website
- ✨ Real-time clock display on homepage
- ✨ Public pages for schedules (Piket, Praktikum, Seminar)
- ✨ Information pages (Dosen, Asisten, Pengurus)
- ✨ Secure admin authentication with JWT
- ✨ Full CRUD admin dashboard
- ✨ Image upload functionality
- ✨ MySQL database with Sequelize ORM
- ✨ REST API backend with Express.js
- ✨ React frontend with Vite
- ✨ Tailwind CSS styling
- ✨ Framer Motion animations
- ✨ Responsive design for mobile and desktop

#### Backend
- Node.js + Express server
- MySQL database integration
- JWT authentication
- Bcrypt password hashing
- Multer file upload
- Input validation
- CORS configuration
- Database seeding

#### Frontend
- React 18 with Vite
- React Router for navigation
- Axios for API calls
- Context API for state management
- Tailwind CSS for styling
- Framer Motion for animations
- React Hot Toast for notifications
- Protected routes for admin

#### Documentation
- Comprehensive README
- Installation guide
- Deployment guide
- API documentation
- Quick start guide
- Contributing guidelines
